# Demos

We prepared online demos for easy access in order to evaluate web-side vulnerabilities:

1. web threat #1: What wallets can be detected through EIP-6963.
2. web threat #2/#3: If the wallet supports `wallet_revokePermission` method.
3. web threat #4: If the wallet's address can be leaked to a never-connected website.

## Links

The demo for web threat #1 and web threat #2,3 can be found at:

https://dappx.weihongw.com/

Source code: `./dappx`

The demo for web threat #4 can be accessed at:

https://wallet-privacy.distriled.dnetcloud.cs.kuleuven.be/

Source code: `./wallet-privacy`

## How to use

Simply follow the instructions on the websites or refer to Experiment 5 and 6 in `ARTIFACT-APPENDIX.md`
